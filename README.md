# SWCAN Reader
This project defines a schematic and pcb of a small single board SWCAN reader using USB CDC for computer communication and a TH8056 as the transciever, utilizing a STM32 MCU as the boards brain. The board includes a SWD header to allow for initial programming, future programming possible through DFU using the bootloader.

This board is in development and may see changes.

### Software Used
* KiCAD 10
* STM32CubeMX
* STM32CubeProgrammer
* CLion
* CMake

The STM32 MCU is a STM32G491KCU6
